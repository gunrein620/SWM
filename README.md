# Dorm Basketball Board

Single-court weekly basketball reservation board built with Next.js App Router and Vercel KV.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Add environment variables:

```bash
KV_URL=...
KV_REST_API_URL=...
KV_REST_API_TOKEN=...
KV_REST_API_READ_ONLY_TOKEN=...
CRON_SECRET=...
```

3. Run locally:

```bash
npm run dev
```
